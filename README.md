Description
-----------
This project represents implementation of Microsoft GWBasic IDE based on the
following technologies:

* **OpenGL/GLUT** (ide);
* **Flex/Bison** (for grammar rules);
* **C** (shell, ide, runtime).

Developed for educational purposes.
